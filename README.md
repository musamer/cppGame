# Knight Academy: C++ Gamified Learning Platform ⚔️

**Knight Academy** (أكاديمية الفرسان) هي منصة تعليمية مبتكرة تهدف إلى تدريس لغة ++C والبرمجة التنافسية للناشئين (12-16 سنة) بطريقة الألعاب (Gamification). تجمع المنصة بين المحتوى التعليمي الرصين وعناصر التحفيز المستمر لتحويل التعلم إلى رحلة بطولية.

## 🚀 المميزات الرئيسية (Core Features)

- **الهيكلية الهرمية (Educational Hierarchy)**:
  - **العوالم (Worlds)**: مسارات كبرى مقسمة حسب مستويات المعرفة (الأساسيات، الخوارزميات، التقنيات، الأشجار).
  - **المراحل (Stages)**: كل درس أو مسألة هي مرحلة مستقلة يجب فتحها بالتسلسل.
- **التدرج المنطقي (Logical Progression)**: نظام يضمن انتقال الطالب من "مرحباً بالعالم" إلى أعقد المسائل الخوارزمية خطوة بخطوة.
- **محرك التصحيح بالذكاء الاصطناعي (AI Correction Engine)**:
  - تقييم الكود برمجياً ومنطقياً باستخدام OpenAI API.
  - تقديم تعليقات تفصيلية للطلاب باللغة العربية.
  - تقييم الصعوبة والتعقيد الزمني (Time Complexity).
- **العناصر التحفيزية (Gamification)**:
  - **نقاط الخبرة (XP)**: يحصل عليها الطالب عند حل المسائل ورفع كفاءة الحل.
  - **المستويات (Levels)**: تزيد رتبة الطالب كلما زادت نقاطه.
  - **الألقاب (Titles)**: ألقاب مميزة (فارس مبتدئ، قائد فرسان، الفارس الأعظم) بناءً على الإنجاز.
- **لوحة تحكم المشرف (Admin Toolkit)**:
  - إدارة المحتوى التعليمي والمراحل.
  - ميزة "تصفير البيانات" لبدء دورات تدريبية جديدة مع الحفاظ على حسابات الطلاب.
- **دعم اللغات (Multi-language Support)**: واجهة كاملة باللغتين العربية والإنجليزية.

## 🛠 التقنيات المستخدمة (Tech Stack)

- **PHP 8.x (MVC Framework)**: هيكلية منظمة وسريعة.
- **MySQL**: قاعدة بيانات قوية لإدارة المستخدمين والتقدم.
- **Tailwind CSS**: تصميم عصري ومتجاوب.
- **Monaco Editor**: محرر كود احترافي (يشبه VS Code) داخل المتصفح.
- **OpenAI API**: لتقييم الكود وتقديم الملاحظات الذكية.
- **Wandbox API**: لتشغيل الكود ومعاينة المخرجات محلياً.

## 📂 هيكلية المشروع (Project Structure)

- `app/`: الكود البرمجي الرئيسي (Controllers, Models, Views, Core).
- `config/`: ملفات الإعدادات والربط مع قاعدة البيانات.
- `public/`: نقطة الدخول الرئيسية، ملفات CSS و JS.
- `learndata/`: المحتوى التعليمي الخام (HTML/Markdown) الذي يتم استيراده للنظام.
- `database/`: سكيما قاعدة البيانات (SQL) والملفات المساعدة.

## ⚙️ التثبيت والتشغيل (Setup)

1. قم برفع المشروع على خادم (Laragon أو XAMPP).
2. أنشئ قاعدة بيانات باسم `cpp_gamified_db`.
3. قم باستيراد ملف `database/schema.sql`.
4. عدل ملف `config/config.php` ببيانات قاعدة البيانات ومفتاح OpenAI الخاص بك.
5. ادخل باسم مستخدم بصلاحية `admin` أو `instructor` للوصول لأدوات التحكم.
